# Manga Hub

漫画投稿・閲覧サイトのプロトタイプです。

## 機能

- **ホーム**: 漫画一覧の表示
- **詳細ページ**: 漫画の詳細とエピソード一覧
- **ビューワー**: 漫画のエピソード閲覧
- **アップロード**: 漫画の投稿（UIのみ）

## 実行方法

開発サーバーを起動するには、以下のコマンドを実行してください：

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いて確認できます。
